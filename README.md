# curso-git-code.education

* Criei um repositório no **gitHub.com**

* Criei uma pasta no meu _local_

* Abri o Git Bash

* Iniciei um repositório com _git init_

* Fiz a sincronização com _git remote add origin https://github.com/thiago14/curso-git-code.education.git_

* Depois o comando _git pull origin master_ para pegar o README.md que havia criado ja no **gitHub** para teste

* Alterei no meu local o arquivo README.md

* Fiz os comando **git** para atualizar o arquivo.
